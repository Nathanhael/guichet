# Workflow: Tessera

## Deployment Lifecycle
**Research -> Strategy -> Execution**

## Execution Cycle
**Plan -> Act -> Validate**

## Critical Mandates
- **DOCKER ONLY**: Never run npm/node commands on the host machine.
- **STRICT B&W**: Never introduce colors, gradients, or shadows.
- **TYPE SAFETY**: 100% tRPC and Drizzle coverage.
- **ZERO MOTION**: UI must remain static.
