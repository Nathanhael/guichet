# Design Spec: Guichet Support App Logo

**Date**: 2026-04-13
**Topic**: Logo Generation (Re-brand)
**Status**: Approved (Autonomous)

## 1. Objective
Create a high-resolution logo for the "Guichet" support application that moves away from the Brutalist aesthetic towards a "warm, human, approachable" identity.

## 2. Visual Identity
- **Style**: Timeless Institutional / Humanist.
- **Palette**:
  - Background: `#FAF9F6` (Cream / Warm Stone)
  - Icon Frame: `#1E293B` (Slate-800 / Deep Navy)
  - Window Glow: `#B45309` (Amber-700) to `#FCD34D` (Amber-300)
  - Wordmark: `#1E293B` (Slate-800)
- **Typography**: Georgia or similar refined Serif (Weight: 700).

## 3. Composition
- **Canvas**: 512x512.
- **Icon**: A small arched window (porthole) with a warm amber light glowing through it.
- **Wordmark**: "GUICHET" centered below the icon with generous letter spacing.

## 4. Assets to Generate
- `client/public/icons/guichet-512.svg`: Scalable vector version.
- `client/public/icons/guichet-512.png`: 512x512 raster version for manifest/favicons.
