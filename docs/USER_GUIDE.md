# User Guide: Tessera Platform (Clean Slate)

Welcome to Tessera. This guide explains the core chat functionality and the strict monochrome interface.

---

## 1. Roles & Responsibilities

The platform is designed around 4 distinct roles:

1.  **Platform Operator**: Global administrator. Manages the Partner ecosystem.
2.  **Partner Admin**: Manages organization-specific settings and members.
3.  **Support Specialist**: Handles incoming tickets and communicates with agents.
4.  **Agent**: Creates tickets and communicates with support on behalf of customers.

---

## 2. Using the Support Workspace

The Support Specialist workspace is optimized for high-performance communication.

### Live Queue
- Tickets appear in the sidebar and are sorted by wait time.
- Join a ticket to start a real-time conversation.
- Use **Shortcuts** for common procedures.

### Static Interface
- All animations and transitions have been removed for immediate response.
- The UI is strictly black and white to ensure maximum contrast and reduced eye strain.

---

## 3. Accessibility Features

Tessera prioritizes readability through high-contrast design and specialized modes.

### Dyslexic Mode
- Switch to the **Lexend** font family, specifically designed to reduce reading errors.
- Increased line height for better scanability.

### Bionic Reading
- Highlighting fixation points (the first few letters of words) to help process text faster.

### High Contrast Standard
- The entire application uses a strict Black & White palette. Toggle **Dark Mode** to invert the palette while maintaining the high-contrast standard.

---

## 4. Multi-Tenant Switching

If you are a member of multiple organizations, use the **Partner Switcher** in the top navigation to jump between workspaces.
