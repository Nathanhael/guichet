| `roles.ts` | Role hierarchy and permission checks (agent < support < admin < platform) |

| `sla.ts` | SLA enforcement with per-department config and alerting |

| `systemMessage.ts` | System message generation (join/leave/transfer notifications) |

| `webhookDispatch.ts` | Webhook event dispatch to partner-configured endpoints (HMAC-signed) |
