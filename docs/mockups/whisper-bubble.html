Deltas vs the current implementation
· Label renamed to Whisper (matches whisperMode in ComposeArea; reads as "private between staff" rather than "unilateral note"). Missing internal_note i18n key replaced with whisper_label.
· Sender shown on whispers. The current MessageBubble branch !isMine && !isWhisper && isGroupStart actively hides the name for whispers — that's a bug. New render: when a whisper's sender differs from the previous whisper's sender (or it's the first in a run), show Whisper · Lucas in the header. Subsequent whispers from the same sender collapse to just the body (same as normal message grouping).
· Multi-sender runs. Consecutive whispers from different senders each get their own header row, so Sophie's reply is attributed distinctly inside the same run.
· Dashed purple separators bracket the whisper run: — Whisper — on the way in, — End whisper — on the way out. The run is visibly lifted out of the main agent↔support thread. Only emitted when the previous message wasn't also a whisper (start) and the next message isn't a whisper (end) — no separator between consecutive whispers inside a run.
· Italic + dimmed body (V3 from the earlier mockup). Italic = aside convention; dimmed = lower priority than the main thread.
· Lock icon next to the label reinforces "staff-only".
