Tessera — Compose area redesign

Unify the format toolbar and compose row into one bordered box, swap text labels for Lucide icons, clean the placeholder, and make whisper mode visually load-bearing.

Before — detached toolbar, text-label format buttons, placeholder hint

After — unified box, icon format strip, clean placeholder, send with ⏎ hint

Format strip is now part of the compose box (thin divider separates it from the main row). Format buttons use Lucide icons (Bold, Italic, Code, Quote, List). The attach tip moved from the placeholder to a dimmed hint row on the right of the format strip. Send button gained an ⏎ kbd mark so first-time users know Enter sends.

Ctrl+V paste ⏎ send

After — whisper mode active

Purple border around the entire compose box (not just a background tint), a slim purple banner above with Lock icon + "Whisper · only visible to support staff", the whisper toggle button flipped to the filled purple state, and the send button also turns purple so you can't accidentally send a whisper thinking it's going to the customer.

My gut ranking: MessageSquareLock reads "locked message" most literally, Users says "between us staff" most accurately, and EyeOff is the safest "not visible to the customer". Ghost is playful but a touch too cute for a brutalist tool. VolumeX and Shield are available but less precise.

Summary of deltas
· Unified compose box. Format toolbar, action buttons, textarea, and send are one bordered unit with internal dividers, not three detached pieces.
· Lucide icons for format. Bold / Italic / Code / Quote / List as real icons, 28×28 buttons with hover state. Text labels (B I </> "" •) are gone.
· Clean placeholder. Type a message… (or Type a private note for support staff… in whisper mode). The "Ctrl+V paste" hint moved to the right side of the format strip as a dimmed kbd-style mark.
· Send button with ⏎ hint. Label + a faint ⏎ kbd mark. First-time users learn Enter-to-send by osmosis.
· Lock icon for the whisper toggle (replaces EyeOff). Matches the icon on the whisper bubble label from the previous pass — one visual vocabulary for "private".
· Whisper mode is unmissable. Purple border around the whole compose box, purple banner row above the format strip with Whisper · only visible to support staff, purple send button labelled Whisper. You cannot accidentally ship a whisper thinking it's a reply.

Out of scope for this mockup (but adjacent): the AI improve revert bar, the reply-to preview, the file-attachment preview strip, and the canned-response picker all sit above the compose box and are unchanged.
